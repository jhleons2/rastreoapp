    ⚡ SOLUCIÓN RÁPIDA PARA CONECTAR POSTGRESQL

PROBLEMA: Railway no comparte automáticamente las variables

SOLUCIÓN: Agregar variables manualmente

═══════════════════════════════════════════════════════════════

PASO 1: Copiar valores de PostgreSQL
──────────────────────────────────────

En Railway Dashboard:

1. PostgreSQL → Variables
2. Copia estos valores (haz "Reveal" en cada una):

   PGHOST → [valor]
   PGPORT → [valor]
   PGUSER → [valor]
   PGPASSWORD → [valor]  
   PGDATABASE → [valor]

PASO 2: Agregar a rastreoapp
───────────────────────────────

1. rastreoapp → Variables
2. Click "+ New Variable"
3. Agrega CADA una de las 5 variables con sus valores

PASO 3: Redeploy
───────────────────

1. Deployments → "..." → Redeploy
2. Espera 1-2 minutos

═══════════════════════════════════════════════════════════════

ALTERNATIVA: Usar Variable Reference
─────────────────────────────────────

1. PostgreSQL → Variables
2. Busca banner morado
3. "Variable Reference" → "+ Add Reference"
4. Selecciona "rastreoapp"
5. Railway las compartirá automáticamente
